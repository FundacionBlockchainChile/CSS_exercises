So What Actually Happends?
  1. The browser goes to the DNS server DNS server and finds the real address of the server that the website lives on.
  2. The browser sends an HTTP request message to the server asking it to send a copy of the website to the client.
  3. If the server approves the client request, the server sends the client a "200 OK" message , wich means "Permision granted! Here you go!", and then starts sending the website´s files to the browser as a series of small chunks.
  4. The browser assembles the small chunks into a complete website and displays it to you!
